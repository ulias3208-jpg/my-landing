# CLAUDE.md — Telegram Mini App

## Структура файлов

```
tg-app/
├── index.html           — Точка входа, подключает SDK + стили + скрипт
├── style.css            — Все стили (темы, экраны, карточки, анимации)
├── app.js               — Вся логика (навигация, рендер, отправка заявок)
├── data/
│   └── services.json    — Каталог услуг (здесь менять данные!)
└── CLAUDE.md            — Этот файл
```

## Навигация между экранами

```
Главная  →  Детали услуги  →  Заявка  →  Попап «Готово»
  (main)      (detail)        (order)     (overlay)
```

- **Вперёд**: slide вправо (300ms)
- **Назад**: slide влево (300ms) через BackButton API
- Стек экранов хранится в `screenStack[]`
- Нет hash-роутера — управление через JS

## Где менять данные

### Услуги, цены, описания
Файл: `data/services.json`

Каждая услуга:
```json
{
  "id": "sites",
  "icon": "🖥",
  "name": "Сайты и лендинги",
  "price_label": "от 5 000 ₽",
  "description": "Описание...",
  "includes": ["Пункт 1", "Пункт 2"],
  "variants": [
    { "name": "Лендинг", "price": "от 5 000 ₽", "days": "3–5 дней" }
  ],
  "review": { "text": "Отзыв...", "author": "Имя" }
}
```

### Telegram-бот (отправка заявок)
Файл: `app.js`, объект `CONFIG` в начале файла:
```javascript
const CONFIG = {
  botToken: '...',
  chatId: '...',
  telegramLink: 'https://t.me/Samarina_Yuliya',
};
```

### Стили и цвета
Файл: `style.css`, CSS-переменные в `:root`

Цвета берутся из Telegram темы автоматически через `var(--tg-theme-*)`.
Фирменные цвета (accent, success, error) заданы отдельно.

## Telegram Web App API — что используется

| API | Где | Зачем |
|-----|-----|-------|
| `tg.ready()` | app.js, инициализация | Сообщает TG что приложение готово |
| `tg.expand()` | app.js, инициализация | Разворачивает на весь экран |
| `tg.BackButton` | Навигация | Нативная кнопка «Назад» |
| `tg.MainButton` | Детали, Заявка | Кнопка CTA внизу экрана |
| `tg.HapticFeedback` | Все интерактивные элементы | Вибрация при нажатии |
| `tg.openTelegramLink()` | Кнопка «Написать» | Открывает чат без выхода из TG |
| `tg.initDataUnsafe.user` | Приветствие, заявка | Имя и username пользователя |
| `tg.themeParams` | CSS-переменные | Автоадаптация к теме |

## Правила редактирования

- Без фреймворков — только Vanilla JS
- Изображения — WebP, до 150KB
- Минимум 48px для тач-зон
- Язык: русский
- Для обхода кэша TG: версионировать файлы (`app.js?v=2`)
