## 1. Цель сайта и целевая аудитория

### 1.1 Цель сайта
**Основная:** Превратить случайного посетителя в заявку — написать в Telegram, заполнить форму или позвонить.

**Вторичные:**
- Показать Юлию как доступного и результативного цифрового мастера
- Развеять страхи («долго», «дорого», «сложно разобраться»)
- Позиционировать нейросессии и ролики как отдельный ценный продукт

### 1.2 Целевая аудитория

**Первичная — покупатели продуктов (лендинги, сайты, Mini Apps, агенты):**

| Параметр | Описание |
|---|---|
| Кто | Малый бизнес, самозанятые, предприниматели |
| Потребность | Быстро и недорого выйти в онлайн или автоматизировать задачу |
| Боль | Агентства дорогие, фрилансеры пропадают, не знают куда идти |
| Цена | 5 000 – 40 000 ₽ — доступный сегмент, решение принимается быстро |
| Как решают | Ищут через рекомендации, Telegram-каналы, Instagram |
| Вопрос в голове | «Она уже делала что-то похожее на моё — и что вышло?» |

**Вторичная — покупатели знаний (нейросессии, ролики, курсы):**

| Параметр | Описание |
|---|---|
| Кто | Люди любых профессий, которые хотят освоить AI |
| Потребность | Понять нейросети без технического бэкграунда |
| Боль | Много информации, непонятно с чего начать, страшно |
| Что хотят | Простое объяснение + практика, которая даст результат быстро |
| Пример | Врач из отзыва — освоила AI для работы и экономит часы каждый день |

### 1.3 Ключевой инсайт из экспертной оценки
> Клиент нанимает не технологию — он нанимает **человека**, который решит его задачу. Юлия должна звучать как партнёр, а не как сервис-поставщик.

### 4.2 Типографика

| Шрифт | Использование | Характеристики |
|---|---|---|
| **Space Grotesk** | Заголовки (H1–H3), логотип | 700, `letter-spacing: -0.02em` |
| **Inter** | Основной текст | 400/500, `line-height: 1.7` |

**Размеры (clamp-адаптивные):**
- Hero H1: `clamp(3.5rem, 9vw, 6rem)`
- Section H2: `2.2rem`
- Body: `1rem / 0.9rem`

### 4.3 Настроение / Mood

**Ключевые слова:** тёмный, умный, живой, доступный, современный, AI-эстетика

**Визуальные сигналы «работаю с AI»:**
- Анимированные орбы из градиентов за hero (фиолетово-розовый + голубой)
- Dot-grid паттерн (точечная сетка, 7% прозрачность)
- Glassmorphism карточки (`backdrop-filter: blur(8px)`)
- Кастомный курсор (точка + следующий круг)

**Принцип контраста:**
- Тёмный фон + яркие акценты = профессионализм
- Живые градиенты = энергия и современность
- Сдержанность = не кричащий дизайн, а умный

## 5. Обязательные элементы

### 5.1 Кнопки

| Вариант | CSS-класс | Применение |
|---|---|---|
| Основная | `.btn.btn--primary` | Главные CTA |
| Прозрачная | `.btn.btn--ghost` | Вторичные действия |
| Маленькая | `.btn.btn--sm` | Nav CTA |
| На всю ширину | `.btn.btn--full` | Кнопка формы |

**Микроинтеракции:**
- `hover`: `translateY(-2px)` + `box-shadow` с акцентным цветом
- `active`: `scale(0.97)` — физически правдоподобный feedback

### 5.2 Форма (Contact)

- Максимум 3 поля (Имя, Email, Сообщение)
- Валидация: имя + email обязательны, email по регулярному выражению
- Состояния: `loading` (disabled кнопка + «Отправляем...»), `success` (зелёное сообщение), `error` (красное сообщение)
- Сообщение исчезает через 5 секунд

### 5.3 Анимации

| Элемент | Анимация | Триггер |
|---|---|---|
| Все секции | `.reveal` — `opacity: 0 → 1`, `translateY: 28px → 0` | IntersectionObserver |
| Карточки в сетке | Stagger delay 0.1s / 0.2s | При появлении |
| Счётчики в Hero | EaseOut cubic за 1400ms | IntersectionObserver |
| Орбы в Hero | Float-анимация (18s / 22s loop) | CSS keyframes |
| Parallax орбов | `translateY` пропорционально `scrollY` | Scroll event (passive) |
| FAQ | `max-height: 0 → 200px` | Клик |
| Кастомный курсор | Dot instant + Outline spring lerp (ease: 0.12) | mousemove |
| Nav | `.scrolled` класс при `scrollY > 40px` | Scroll event |

### 5.4 Кастомный курсор

- `#cursorDot` — 8px, instant follow
- `#cursorOutline` — 36px, spring lag (ease factor 0.12)
- Expand при hover на `a, button, input, textarea`
- Скрыт на touch-устройствах (`@media (hover: none)`)

### 5.5 FAQ Аккордеон

- Один открытый элемент одновременно (остальные закрываются)
- ARIA: `aria-expanded="true/false"` на кнопках
- Шеврон вращается 180° при открытии

### 5.6 Мобильная версия

- Burger-меню: 3 линии → X при открытии
- Fullscreen overlay при открытом меню (body overflow hidden)
- Все grid переходят в 1 колонку на 768px
- Кнопки Hero — на всю ширину на 480px

## 9. Копирайтинг — принципы голоса Юлии

**Tone of Voice:** дружелюбный, прямой, без пафоса, уверенный, живой

**Работающие формулировки:**
- «Быстро, красиво и с душой» ← её собственный стиль
- «Вайбкодер» ← уникальный идентификатор, запоминается
- «С помощью AI» ← показывает инструментарий, не пугает технологиями
- «Работаю по всему миру» ← снимает географическое ограничение

**Избегать:**
- «Cutting-edge», «инновационный», «революционный»
- «AI-powered» как заголовок (слишком затёрто в 2025)
- Сложный технический жаргон — аудитория не технари

**Соотношение «вы/ваш» к «я/мой»:** минимум 3:1 в пользу «вы»
> Сейчас страница немного смещена в «я». При правках текстов держать это соотношение.
